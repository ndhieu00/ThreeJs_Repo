# ThreeJS demo

## Requirements

- Node.js

## Setup

### 1. Run Frontend

From root project

```bash
cd frontend
npm install
npm run dev
```

### 2. Run Backend

From root project

```bash
cd backend
npm install
npm start
```

### 3. Configure Frontend with Backend Port

Check the backend port in use.
Update the frontend/.env file with the correct backend port, e.g: VITE_API_URL=http://localhost:3000
